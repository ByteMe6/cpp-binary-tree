# cpp-binary-tree

Реализация бинарного дерева поиска на C++.

[English](./README.md)

## Структура

```
treeStruct.h      — структура Tree и объявления функций
addNewNode.cpp    — рекурсивная вставка
tree_search.cpp   — рекурсивный поиск с записью пути
print_tree.cpp    — вывод дерева в pre-order
print_tree2.cpp   — вывод дерева «на боку»
main.cpp          — демо: вставка 10 случайных значений, поиск 5
```

## Как работает

- Вставка: значения меньше текущего узла — влево, больше — вправо
- Поиск: обходит дерево и записывает каждый шаг (`"left"` / `"right"`)
- Возвращает `-1` и `{"nothing"}` если значение не найдено

## Сборка

```bash
cmake -B build
cmake --build build
./build/binaryTree
```

Требуется C++26 и CMake ≥ 4.3.
