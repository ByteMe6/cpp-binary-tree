# cpp-binary-tree

Реализация бинарного дерева поиска на C++.

[English](./README.md)

## Структура

```
tree.h       — структура Tree и объявления функций
tree.cpp     — вставка, поиск и вывод
main.cpp     — демо: вставка 10 случайных значений, поиск 5
```

## Как работает

- Вставка: значения меньше текущего узла идут влево, больше — вправо
- Поиск: обходит дерево и записывает каждый шаг (`"left"` / `"right"`)
- Возвращает `-1` и `{"nothing"}` если значение не найдено

## Сборка

```bash
cmake -B build
cmake --build build
./build/binaryTree
```

Требуется C++26 и CMake ≥ 4.3.
